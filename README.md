# scala-cli-maven-failsafe-spring-pmd-cucumber-mockito-junit-car-data

## Description
A POC for maven app using junit5
and cucumber and mockio framework
 with pmd
and failsafe plugin.

## Tech stack
- scala
- maven
	- spring
	- mockito
  - junit5
  - failsafe
  - pmd
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
